# 自动化计算时间

`./time_eval.py`

## 文件名字格式

`p<problemid>.[????.]<ext>`

如 `p101.speed3.py` `p233.cpp` `p1.rs`

## 目前超时1min

自动掐断+ 设置巨大时间 1000000

## 支持格式

目前支持了py(python3) cpp(clang++ std=gnu17) rs(rust) go(go) js(node) txt(plain text) sh(shell)
